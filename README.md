# IT-Warehouse

<img width="1027" height="737" alt="raid-calculator" src="https://github.com/user-attachments/assets/dd3efba8-ef4c-4942-a93c-5a4d435209ee" />


Disk sayısı, disk boyutu ve RAID seviyesi seçilerek kapasite, hız kazanımı ve hata toleransını hesaplayan **basit, özgür ve açık kaynak bir RAID hesaplama aracı**.

Desteklenen RAID seviyeleri:

| Standart | Gelişmiş |
|----------|-----------|
| RAID 0 | RAID 1E |
| RAID 1 | RAID 5E |
| RAID 5 | RAID 5EE |
| RAID 6 | RAID 50 (RAID 5 + 0) |
| RAID 10 | RAID 60 (RAID 6 + 0) |

---

## Özellikler

-Kapasite (TB) hesaplama  
-Hız kazancı (okuma/yazma) metinsel tahmin  
-Hata toleransı (kaç disk arızasına dayanır)  
-RAID 50 / RAID 60 için otomatik grup yerleşimi  
-Sağ panelde seçilen RAID yapısının kısa açıklaması  
-Modern tasarımlı Tkinter arayüzü  
-Ek kütüphane gerektirmez – sadece Python 3 yeterli
